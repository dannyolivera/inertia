## About

InertiaJS application developed with the following technologies.

- Laravel [10]
- VueJS [3]
- Tailwindcss [3]

## Run composer
``` bash
composer install
```
## Run artisan
``` bash
php artisan serve
```
## Run npm
``` bash
npm install
```
``` bash
npm run dev
```
## Run migrations
``` bash
php artisan migrate --seed
```
